# PaDrinks Website

Sitio web oficial de PaDrinks con política de privacidad.

## URLs

- **Sitio principal:** https://padrinks.com
- **Política de privacidad:** https://padrinks.com/privacy-policy

## Despliegue

Este sitio está desplegado en Vercel y conectado al dominio padrinks.com

Para actualizar:
1. Edita los archivos HTML
2. Haz commit y push
3. Vercel despliega automáticamente

## Archivos

- `index.html` - Página principal
- `privacy-policy.html` - Política de privacidad (requerida para Play Store/App Store)
- `vercel.json` - Configuración de Vercel
